# PONG GAME

> This game has a pong, represented by the white circle and two paddles, represented by white rectangule on both sides of the screen.

## How to Play
The game is played by controlling either of the paddles to interact with the pong.
- The pong bounces of the upper and lower ball but not when almost parallel to the net
- Pong past the screen is a point for the opponent of the side which the pong overshoots

### Controls
**Arrow Keys** - for the right paddle
```
    "Up" key to move up
    "Down" key to move down
```
**Letter Keys** - for the left paddle
```
    "w" key to move upwards
    "s" key to move downwards
```
